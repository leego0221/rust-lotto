# Rust로 로또 발매기 구현하기

## 개요

콘솔 기반 로또 발매기 프로그램입니다.

사용자가 로또 구입 금액을 입력하면 금액만큼 로또를 발행합니다.
당첨 번호와 보너스 번호를 입력하면 각 로또의 등수를 계산하고, 구입 금액에 따른 수익률을 계산합니다.

각 단계마다 발행된 로또 번호와 당첨 내역, 수익률을 콘솔로 확인할 수 있습니다.

## 구현할 기능 목록

### 입력

- 구입 금액 입력
- 당첨 번호 입력
- 보너스 번호 입력

### 출력

- 구매 개수 출력
- 구매한 로또 리스트 출력
- 구매한 로또 리스트 중 몇 개 일치했는지 출력
- 수익률 출력

### 기능

- 구입 금액에 해당하는 만큼 로또 발행하기
- 1개의 로또를 발행할 때 중복되지 않는 6개의 숫자 뽑기
- 번호 일치 여부에 따라 등수 매긴 뒤 내부에 저장하기
- 등수에 따라 당첨 금액을 부여하고, 총 구매 금액에 따른 수익률 계산하기

## 예외 케이스

### 입력

- 공백 계열은 아닌지 검증
- 구입 금액과 보너스 번호의 경우 Integer 타입인지 검증
- 당첨 번호의 경우 쉼표 기준으로 잘 구분되는지 검증

### 구입 금액 입력

- 1,000원 이상인지 검증
- 1,000원 단위인지 검증

### 당첨 번호 입력

- 번호가 총 6개인지 검증
- 번호가 중복되지 않는지 검증
- 번호가 1에서 45 사이인지 검증

### 보너스 번호 입력

- 번호가 1에서 45 사이인지 검증
- 당첨 번호 6개와 중복되지 않는지 검증

### 잘못된 값 입력

- 예외 발생 및 그 부분부터 입력 다시 받기
